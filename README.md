# A Benchmark Dilemma 

<a href="https://chc.au.dk"><img src="https://github.com/centre-for-humanities-computing/intra/raw/main/images/onboarding/CHC_logo-turquoise-full-name.png" width="25%" align="right"/></a>

This code accompanies a research project on benchmarking transformer models for art historical research. It implements different benchmarking tasks, including [1] training lightweight projection head on top of frozen, pretrained transformer backbones via contrastive triplet learning in a naturally occuring, data scarce scenario, [2] different image classification tasks based on wikiart and wikidata, including augmented images for different image qualities and [3] comparing an object detection model (SAM3) to human performance. For more details, see the README files in the respective subfolders.
